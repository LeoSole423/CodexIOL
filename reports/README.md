# Reports Directory Policy

Este directorio contiene reportes generados del portafolio.

Regla del repositorio:
- Se versiona solo la estructura de carpetas (`.gitkeep` y este `README.md`).
- No se versionan reportes reales con informacion personal.

Subcarpetas esperadas:
- `reports/analisis/`
- `reports/macro/`
- `reports/rebalance/`
- `reports/rebalance/archive/`
- `reports/dca/`
- `reports/latest/`
